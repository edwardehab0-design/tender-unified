# لوحة العمليات

المدخل التشغيلي الحالي هو `index.html` مع `tasks.js` و`tasks.css`.

`app.js` و`db.js` يمثلان مسارًا أقدم للوحة العمليات ولا يتم تحميلهما من `index.html` حاليًا. لا تعدل هذا المسار إلا إذا كان الهدف هو إعادة تفعيله أو دمجه بشكل صريح.

مصادر البيانات المستخدمة حاليًا:

- `employees.json`: هيكل الأقسام والموظفين.
- `../tenders/data.json`: قائمة المناقصات الجارية والمقدمة.
- `localStorage`: تخزين المهام محليًا عند عدم وجود جلسة Supabase.
- `ops_state` في Supabase: تخزين مشترك عند توفر إعدادات Supabase وجلسة دخول.
